Опции elFinder

elFinder может быть настроен 
	1 при инициализации:
		$(selector).elfinder({
			option : value
		})
		
	2 до инициализации и после подключения файлов elFinder
		
		можно переопределить все параметры:
		var opts = {
			option1 : value1,
			option2 : value2
			... etc
		}
		
		elFinder.prototype.options = $.extend({}, elFinder.prototype.options, opts)
		
		или только отдельные опции:
		
		elFinder.prototype.options.option1 = value2;
		
		

- url                   : (String) URL коннектора. Единственный обязательный параметр
- lang                  : (String) Язык интерфейса. Двухбуквенное обозначения языка. Требует подключения файла i18n/elfinder.Обозначение_языка.js
- cssClass              : (String) Дополнительный css класс для контейнера
- wrap                  : (Number) Кол-во символов на которые разбивается длинные имена файлов в режиме отображения "иконки". По умолчанию: 14. 0 - не разбивать имена файлов
- places                : (String) Название папки избранное. Локализуется. По умолчанию - "Places". Укажите "", чтобы не использовать places
- placesFirst           : (Boolean) Поместить places перед деревом файлов в панели навигации? По умолчанию - true
- editorCallback        : (Function||null) Функция обратного вызова для wswing редактора (CKEditor, TinyMCE, elRTE и тд). Получает один аргумент (String) - URL выбранного файла.
- closeOnEditorCallback : (Boolean) Закрыть редактор после вызова editorCallback? По умолчанию - true. (Бессмыслено для CKEditor, TinyMCE)
- view                  : (String) Внешний вид текущей директории по умолчанию. Возможные значения: icons||list. По умолчанию - icons.
- width                 : (Number||String) Ширина окна файлового менеджера. По умолчанию не задана, ширина указана в css файле
- height                : (Number) Высота окна файлового менеджера. По умолчанию не задана, указана в css файле
- disableShortcuts      : (Boolean) Отключить сочетания клавиш (кроме стрелок и пробела). По умолчанию - false.
- rememberLastDir       : (Boolean) Открывать последнюю посещенную директорию после релоада страница или закрытии/открытии браузера
- cookie                : (Object) Параметры cookie (вид текущей директории, последняя открытая директория, places). По умолчанию: {expires : 30, domain  : '', path    : '/', secure  : false}
- toolbar               : (Object) Группы кнопок на панели инструментов. По умолчанию: все доступные кнопки/команды. Для настройки - см соответствующий раздел документации.
- contextmenu           : (Object) Содержимое контекстного меню. По умолчанию: все доступные команды. Для настройки - см соответствующий раздел документации.
- dialog                : (Object) Опции диалогового окна jQueryUI dialog, в котором будет открыт elFinder. Если опция docked == true - elFinder будет открыт на странице с возможностью "отстегивания" в диалоговое окно с указаными параметрами
- docked                : (Boolean) Разрешить "отстегивание" elFinder в диалоговое окно (может упоминаться в документации, как docked mode)
- autoReload            : (Number) Количество минут, через которое содержимое текущей директории и дерева директории обновляется. По умолчанию: 0 (отключено)
