Список опций коннектора:

- root          (String) Путь к корневой директории
- URL           (String) Урл корневой директории
- rootAlias     (String) Алиас для корневой директории
- disabled      (Array) Список отключеных команд
- dotFiles      (Boolen) Показывать файлы, начинающиеся с точки
- dirSize       (Boolen) Считать размер, занимаемый директориями
- fileMode      mode для новых файлов
- dirMode       mode для новых директорий
- mimeDetect    (String) способ определения mime-типов файлов (php коннектор только. варианты значений: finfo, mime_content_type, linux (file -ib), bsd (file -Ib), internal (по расширению))
- uploadAllow   (Array) Список разрешенных типов файлов для загрузки. Можно задать точный mimetype (image/jpeg) или группу типов (application)
- uploadDeny    (Array) Список запрещенных типов файлов для загрузки
- uploadOrder:  (Array) Порядок применения правил
	- allow,deny  Только то, что разрешено, кроме того, что запрещено. (AND)
	- deny,allow  То, что не запрещено или разрешено. (OR)
- imgLib        (String) библиотека для создания превьюшек (только php коннектор. Варианты: imagick, mogrify, gd). Если не указан, пытается определить доступную.
- tmbDir        (String) Папка для превьюшек. Если не задана - превьюшки не будут создаваться
- tmbCleanProb  (Int) Частота очистки папки с превьюшками. Варианты - от 0 до 200. 0 - не очищать, 200 - при каждом запросе инициализации файлового менеджера
- tmbAtOnce     (Int) Сколько превьюшек создавать в одном фоновом запросе. По умолчанию - 5
- tmbSize       (Int) Рамер превьюшек в px
- fileURL       (String) Отдавать URL файлов клиенту
- dateFormat    (String) Формат даты изменения файлов. По умолчанию "j M Y H:i"
- logger        (Object) Объект-логгер
- defaults      (Array) Права доступа к файлам/папками по умолчанию. Значение по умолчанию 
	: array(
		'read'   => true,
		'write'  => true,
		'rm'     => true
	)
- perms         (Array) Права доступа к папкам/файлам. Подробнее см соответствующий раздел
- debug         (Boolean) Отсылать дебаг клиенту
- archiveMimes  (Array) Список типов архивов, разрешенных для создания. Если не задан, будут разрешены все доступные типы
- archivers     (Array) Информация об архиваторах. Если не задана, коннектор попытается найти и использовать все доступные архиваторы.

Пример:
	 'archivers' => array(
	 	'create' => array(
	 		'application/x-gzip' => array(
	 			'cmd' => 'tar',
	 			'argc' => '-czf',
	 			'ext'  => 'tar.gz'
	 			),
			'application/x-bzip2' => array(
	 			'cmd'  => 'tar',
	 			'argc' => '-cjf',
	 			'ext'  => 'tar.bz'
	 			)
	 		),
	 	'extract' => array(
	 		'application/x-gzip' => array(
	 			'cmd'  => 'tar',
	 			'argc' => '-xzf',
	 			'ext'  => 'tar.gz'
	 			),
	 		'application/x-bzip2' => array(
	 			'cmd'  => 'tar',
	 			'argc' => '-xjf',
	 			'ext'  => 'tar.bz'
	 			)
	 		)
	 	)
